Digital Etch-a-Sketch

How to use:

Hover your mouse over the squares in the grid to draw along a continuous path, much like a classic Etch-a-Sketch.

The buttons on the far right of the screen allow you to:

  • Erase the current sketch
  • Change the grid resolution (specify more or fewer boxes inside the main drawing area)
  • Experiment with the color interaction used during drawing

Primary technologies demonstrated: JavaScript DOM Manipulation, Flexbox.
